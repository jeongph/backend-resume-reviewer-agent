# backend-resume-reviewer-agent

JVM/Spring 백엔드 이력서를 시니어 개발자 관점에서 리뷰해주는 Claude Code 에이전트 플러그인입니다.

## 설치

```
/plugin marketplace add jeongph/backend-resume-reviewer-agent
/plugin install backend-resume-reviewer-agent
```

## 사용법

Claude Code에서 자연스럽게 말하면 됩니다:

```
Spring 백엔드 이력서 리뷰해줘
Review this resume from a Spring backend perspective
스프링 리뷰어한테 이력서 검토 맡겨줘
```

이력서 언어에 맞춰 자동으로 응답합니다. 특정 언어를 원하면 직접 요청할 수도 있습니다.

## 평가 요소

- JVM/Spring 백엔드 포지션 부합도
- 기술적 깊이와 문제 해결 접근
- 기술 선택의 이유와 트레이드오프
- 이력서 가독성과 구조
- 팀 적합성
